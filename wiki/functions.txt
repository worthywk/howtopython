____________________________________
~~ basic functions
____________________________________

input() example: input("Put something here")
print() example: print("Welcome to Hell")

____________________________________
~~ type functions
____________________________________

int()
example: int("5") # return 5

float()
example: float(5) # return 5.0

str()
example: str(5.5) # return "5,5"

type() - get type
example: type("5") # return string
_____________________________________

round(a, b)
example: round(5.555, 1) # return 5.5

_____________________________________
~~ loop functions
_____________________________________

range(a,b); range(a,b,c) - from "a" to "b" with "c" steps

examples: for i in range(1,5); for i in range(1,10,2)

____________________________________
~~ list functions
____________________________________

list() - create new list
examples: list(range(1,10)) # 1,2,3,4,5,6,7,8,9
